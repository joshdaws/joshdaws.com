# JoshDaws.com

Single-page personal brand hub. Static HTML, no build step, deployed to Vercel.

## Design System

- **Palette:** Deep black (#0d0d0d) background, warm off-white (#f0ece4) text, gold (#c8a86e) accent
- **Typography:** Inter, name at 800 weight, tagline uppercase with wide tracking
- **Motif:** Diagonal stripe texture via CSS repeating-linear-gradient, subtle gold radial glow at top
- **Cards:** Two tiers — primary (YouTube, X, Substack, Taking Dominion) and secondary (side projects). Hover lifts with gold border glow.
- **Responsive:** Single-column stacked cards on mobile, 2-column grid on desktop
- **Footer:** "Soli Deo Gloria"
